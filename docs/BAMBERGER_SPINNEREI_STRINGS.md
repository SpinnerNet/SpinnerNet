## Integration Notes

1. **Prefix Convention**: Alle Bamberger Spinnerei-spezifischen Strings beginnen mit `BambergerSpinnerei_` um Konflikte zu vermeiden

2. **XML Encoding**: Beachten Sie die korrekte Kodierung von Sonderzeichen:
   - & → &amp;
   - < → &lt;
   - > → &gt;

3. **Struktur**: Die Strings folgen der Seitenstruktur:
   - Navigation
   - Hero/Landing
   - Pillars (Säulen)
   - Projects
   - CTAs
   - Footer

4. **Verwendung in Razor**:
   ```razor
   @Model.GetText("BambergerSpinnerei_Hero_Title")
   ```
